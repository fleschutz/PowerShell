Mega Collection of PowerShell Scripts
=====================================

**Contains 500+ free and stand-alone PowerShell scripts for Linux, macOS, and Windows. Useful on the command-line (CLI), for remote control via SSH, for automation (on startup/login/logoff/daily/hourly/shutdown or via AutoHotkey/Jenkins/etc.), for context menus, for voice commands (see [talk2windows](https://github.com/fleschutz/talk2windows)), or simply to learn PowerShell. All scripts are located in the ðŸ“‚[scripts](scripts/) subfolder and support Unicode - a modern console like *Windows Terminal* is recommended.**
